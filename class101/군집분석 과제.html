Q3 SSE구하세요¶

# 엘보우 기법
import matplotlib.pyplot as plt
def elbow(X):
    sse=[]
    for i in range(1, 11):
        km=KMeans(n_clusters=i, random_state=1)
        km.fit(X)
        sse.append(km.inertia_)
    
    plt.plot(range(1,11), sse, marker='o')
    plt.xlabel('The Number of Clusters')
    plt.ylabel('SSE')
    plt.show()
    print(sse)
elbow(X)

결과 해석¶

  1. 콜린스키 하라비츠 지수를 보았을 때, 군집 간 분산 / 군집 내 분산 의 비율이 군집이 2개일 때 151, 3개일 때 142이다.
  2. 엘보우 기법으로 봤을 때, 2~3개 사이에서 군집 내 SSE가 급격히 작아지고 그 이상의 군집 수에서는 SSE 감소 효과가 작다.
  3. 따라서, SSE 감소 효과와 (군집 간 분산/군집 내 분산) 비율 2가지를 모두 따져봤을 때 적절한 군집 수는 3 이다.
  4. 군집 수 3개로 분석해봤을 때, 독립변수 Assault에 대해서는 3개의 군집으로 잘 묶고 있다.
  5. 나머지 Murder, UrbanPop, Rape 독립번수에 대해서는 군집을 명확하게 묶어주고 있지 않다.
